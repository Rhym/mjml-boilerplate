# Email template

## Uses

1. [Nunjucks](https://mozilla.github.io/nunjucks/)
2. [MJML](https://mjml.io/)

## Setup

1. Clone the repo
2. Run `make install`

## Tasks

| Task      | Description          |
|-----------|----------------------|
| `install` | Install dependencies |
| `build`   | Build dist files     |
| `watch`   | Start webpack server |
